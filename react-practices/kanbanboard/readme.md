1. 설치 
    ```bash
    $ npm i
    ```
2. 설정 
    + babel.config.json
    + webpack.config.js
3. 스크립트 추가하기
    ```json
        "scripts": {
            "start": "npx webpack serve --progress --mode development ",
            "build": "npx webpack"
        },
    ```
4. 실행
    ```bash
    $ npm start
    ```
